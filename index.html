How it works

  1. Upload a chart screenshot (PNG/JPG up to 10MB).
  2. The site sends it to the API (mock now; your LLM later).
  3. The API returns direction, score, TP/SL, pips, and R:R.
  4. Login required to reveal results. Sign up is instant and free.
